# Progress log

A running, date-stamped log of where the project is. Three or four sentences per entry. The value is in the accumulation — three months from now you'll be able to scroll back and see exactly how you got here.

## Format

```
## YYYY-MM-DD
What you worked on. What you finished. What got stuck.
```
